HTML

One of the easiest and most widely used laguages is HTML, or Hypertext Markup Language. Its used as the framework language for creating websites, it can be used to navigate the internet, or even in aiding the development and creation of video games

But how does HTML work? HTML tells web browsers how to structure the web pages you visit. HTML consists of a series of elements, which you use to enclose, wrap, or mark up different parts of content to make it appear or act in a certain way. The enclosing tags can make content into a hyperlink to connect to another page, italicize words, and so on.

lets go over some html basics

Tag Name Description
<p> Paragraph Allows us to make plain text
<div> Content Division allows us to group together elements
<h1-h6> Heading creates headings of varying sizes with h1 being the largest
<a> Anchor lets us link anything between the opening and closing tags
<video> Video You wont believe it but we use this for videos
